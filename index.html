Welcome!


This web-based tool is used to explore the geometry behind the concept of a visibility graph and is a tool we created in order to determine if a graph is the visibility of the graph based on a set of given characteristics. In short, we create a graph-based representation of which vertices of a polygon can "see" each other. To learn more behind the mathematics of visibility graphs, view the Mathematical Background page. If you would like to use our first version of the Visibility Graph visualizer, view the Visualizer page


This tool was developed as part of the Computational Geometry research project in the Purdue Experimental Mathematics Lab (PXML)

We would like to thank our project supervisor Dr. Shuyi Weng, our graduate mentor Leo Shen, and Dr. Thomas Sinclair for their support and for this opportunity
