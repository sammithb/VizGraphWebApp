Visibility Graph Visualizer


Here is the first version (Version 1.0) of our Visibility Graph Visualizer tool made using Mathematica


As of now, our tool can be used for plygons with n = 4, 5, and 6 vertices. In our next version, we aim to improve the UI while also creating the tool in React.
This tool is particularly useful to try and classify different graphs and identify what characteristics in a graph make them visibility graphs of polygons.

NOTE: Please wait for a few seconds for vertices to be moved. Once they settle down you can move vertices and see visibility graphs update in real time.
